🔗 Connect Google Sheets
Follow these steps once to enable live sync between both partners. Your data will save to a shared Google Sheet automatically.
1
Create a Google Sheet — go to sheets.google.com, create a new sheet named Champ Water Data, and share it with your partner.
2
Open Apps Script — in the Sheet, click Extensions → Apps Script. Delete the default code and paste in the script provided by your dashboard builder.
3
Deploy the Script — click Deploy → New Deployment → Web App. Set "Who has access" to Anyone. Copy the deployment URL.
4
Paste the URL below — both partners do this once on their device. Your data will sync automatically going forward.
